# Plot and Caption Review

Use when checking plots, captions, and visual claims.

## Checks

- Plot files exist, are nonempty, and are visually inspectable.
- Plot manifest agrees with the final report.
- Blinded regions remain hidden unless unblinding is explicitly allowed.
- Captions name the scope: prefit, postfit, expected-only, observed, validation, diagnostic, or blocked.
- Captions do not use paper-level terms for diagnostic fallback plots.
- Diagnostic fallback plots avoid wording such as official, combined likelihood, post-fit, exclusion, discovery, or measurement unless that scope is actually allowed.
- Axes, units, legends, and categories match the source artifacts.

## Veto Conditions

- Missing or blank plot used in a final claim.
- Plot reveals blinded data without an unblinding gate.
- Caption/conclusion claims stronger scope than the scorecard or claim classification allows.
- Plot appears to compare incompatible samples, regions, or normalizations without explanation.
